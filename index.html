⚙️ Konfigurasi Database Diperlukan
Tempel Project URL dan Anon Key Supabase di bawah ini.
Kredensial disimpan di browser dan tidak dikirim ke mana pun selain server Supabase Anda.
Cara mendapatkan credentials (gratis):
1. Buka supabase.com → Sign up → New Project
2. Tunggu project siap (~2 menit)
3. Klik Settings (ikon kiri bawah) → API
4. Copy Project URL dan anon public key
5. Jalankan supabase_schema.sql di SQL Editor
💡 Sudah pernah setup sebelumnya?
Jika Anda sudah memasukkan credentials di perangkat ini sebelumnya, data tersimpan di browser. Klik "Sudah dikonfigurasi?" di atas untuk langsung ke halaman login.

Deploy ke Netlify? Anda perlu setup sekali lagi karena localStorage berbeda per domain. Atau hardcode credentials di variabel SUPABASE_EMBEDDED_URL dan SUPABASE_EMBEDDED_KEY di dalam file HTML.
